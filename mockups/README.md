# GURU Financial — Design Mockups

Static HTML mockups for the GURU Financial advisor platform. These files demonstrate the UI direction for each major feature area and can be opened directly in a browser.

## Files

| File | Description |
|------|-------------|
| **mockup-index.html** | Demo landing page with navigation links to each mockup |
| **advisor-brief.html** | Pre-meeting advisor intelligence page — action items, position recommendations, and client context for the Kessler family |
| **balance-sheet-mockup.html** | Full household balance sheet showing liquid assets, investments, real estate, liabilities, and net worth summary |
| **client-overview-v6.html** | Client overview dashboard with household snapshot, activity timeline, and key metrics |
| **allocation-tool-v6.html** | Interactive allocation review tool with current vs. target comparisons, drift analysis, and rebalance controls |
| **allocation-landing-v5.html** | Allocation landing page with portfolio summary cards and navigation into detailed allocation views |

## Design Tokens

The shared design system is defined in `../shared/schema.ts` under the `designSystem` export. Key brand values:

- **Primary navy**: `#183B6E`
- **Background**: `#F0EDE8` (warm parchment)
- **Typography**: DM Sans (body), Playfair Display (headings)

## Usage

Open any `.html` file in a browser to preview. No build step required.
